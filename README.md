# BRAND — магазин функционального фитнеса

Первая версия сайта. Ч/б минимализм, корзина с отправкой в Telegram, каталог на 66+ товаров.

## Как открыть локально

Просто открыть `index.html` в браузере **через локальный сервер** (не двойным кликом — иначе не загрузится `products.json`).

**Способ 1 — Python (есть на любой macOS/Linux):**
```bash
cd site
python3 -m http.server 8000
```
Потом открыть http://localhost:8000

**Способ 2 — VS Code:**
Установить расширение "Live Server", кликнуть правой кнопкой на `index.html` → "Open with Live Server"

**Способ 3 — любой хостинг:**
Залить всю папку `site/` на хостинг — сразу заработает.

## Что нужно заменить / доделать

1. **Название "BRAND"** — найди по всем файлам и замени на твоё (поиск по: `BRAND`)
2. **Логотип** — сейчас текстовый. Если будет SVG-логотип, заменю его в хедере и футере
3. **Hero-картинку** — сейчас тёмный градиент с сеткой. Можно поставить видео/фото твоё — меняется в `css/style.css` в блоке `.hero-bg`
4. **Telegram ник** — @xpodium_russia стоит везде, меняется в `js/app.js` (SHOP_CONFIG.telegram) и в HTML-файлах в ссылках
5. **Фото товаров** — сейчас из прайса (маленькие). Когда снимешь свои — заменить файлы в `images/products/`
6. **Тексты "Our Story"** — мои шаблонные, перепиши под себя
7. **Названия некоторых товаров** — я перевёл 60+ штук, но 5-10 могли остаться на англ. (посмотри в каталоге)
